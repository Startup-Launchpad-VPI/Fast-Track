# Fast-Track

Public resources supporting the **Fast-Track programs** run by the
[EPFL Startup Launchpad](https://startup.epfl.ch) (VPI).

This repository gathers the shared, openly available materials that program
participants, coaches, and partners need: templates, guides, and reference
documents used across Fast-Track cohorts.

All materials in this repository are intended to be public and reusable in the
context of EPFL Startup Launchpad programs. If you're a participant, refer to
your program coordinator for how each resource fits your cohort's timeline.
